## 目的

本文档介绍基于Huawei LiteOS如何移植到第三方开发板，并成功运行基础示例。

## 读者对象

本文档主要适用于Huawei LiteOS Kernel的开发者。
本文档主要适用于以下对象：

- 物联网端软件开发工程师

- 物联网架构设计师
